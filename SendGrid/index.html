SendGrid Plugin Demo

The SendGrid plugin allows you to reliably send email through a highly scalable cloud infrastructure. No local email account is required - your app only needs an internet connection.

IMPORTANT
Make sure you specify the API_USER (Username) and API_KEY (password) parameters in the plugin settings.

Custom plugins don't work in the simulator and the AppBuilder companion app, so inside Telerik AppBuilder choose Run - Build - iOS/Android - App package.

Send a plain text email

If you must, you can send plain text email through SendGrid. Any HTML tags will be escaped in the process.


Send HTML email with attachment
